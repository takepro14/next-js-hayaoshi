# 音声ファイル

このフォルダに以下の音声ファイルを配置してください：

- `bgm.mp3` - ゲーム中のBGM（ループ再生されます）
- `correct.mp3` - 正解時の効果音
- `incorrect.mp3` - 不正解時の効果音

## 音声ファイルの推奨仕様

- **フォーマット**: MP3（互換性が高いため）
- **BGM**: ループ可能な楽曲（30秒〜2分程度）
- **効果音**: 短い音（0.5秒〜2秒程度）
- **ファイルサイズ**: できるだけ小さく（各ファイル1MB以下を推奨）

## 無料の音声素材サイト

- [フリー効果音・無料効果音](https://soundeffect-lab.info/)
- [OtoLogic](https://otologic.jp/)
- [魔王魂](https://maou.audio/)
- [MusMus](https://musmus.main.jp/)

音声ファイルが存在しない場合、アプリケーションは正常に動作しますが、音声は再生されません。
